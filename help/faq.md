# Частые вопросы

{% hint style="info" %}
* **URL для запросов:** `https://proxy.killa.cc/api/v1`
* **Авторизация:** передавайте `Authorization: Bearer <token>` (см. раздел «Авторизация»).
{% endhint %}

### Зачем нужны quote-методы?

Quote возвращает итоговую стоимость **до списания**. Это удобно, чтобы показать цену пользователю и проверить, хватает ли баланса.

### Что будет, если токен утёк?

Считайте токен паролем. Сразу сбросьте его в боте (**Профиль → API → Сбросить токен**). Старый токен станет невалидным.

### Почему получаю 401 Unauthorized?

* Не передан заголовок `Authorization`.
* Неверный формат: нужен `Authorization: Bearer <token>`.
* Токен был сброшен/заменён.

### Почему получаю 400 Bad Request?

Обычно это неверные или отсутствующие параметры запроса. Проверьте поля, типы (string/int/array), обязательные параметры.

### Что означает 5xx?

Ошибка на стороне сервиса или upstream-поставщика. Обычно помогает повторить запрос позже.

### Можно ли безопасно ретраить покупку?

Не ретрайте покупку/продление «вслепую». Если запрос упал по таймауту/сети:

1. Проверьте баланс (возможно списание уже прошло).
2. Проверьте состояние услуги (например `/vpn/info`, `/premium/traffic/balance`).
3. Если сомневаетесь — напишите в поддержку.

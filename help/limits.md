# Лимиты

{% hint style="info" %}
Рекомендованный таймаут клиента: 30–60 секунд

**Ретраи:** только для чтения/quote; покупки и продления — не «вслепую».
{% endhint %}

### Безопасность

* Храните токен как пароль: не коммитьте в репозиторий и не отправляйте в чаты.
* При подозрении на утечку сразу сбрасывайте токен в боте.

### Таймауты и ретраи

* Ставьте клиентский таймаут 30–60 секунд.
* Quote и чтение данных можно ретраить аккуратно.
* Покупки/продления не ретрайте «вслепую»: при сетевой ошибке сначала проверьте факт изменения баланса и состояние услуги.

### Расчёт цены

* Для отображения цены используйте quote-методы: они возвращают итоговую сумму до списания.

### Формат ответа

* Всегда проверяйте поле `ok`.
* При `ok: false` используйте `error.code` и `error.message`.

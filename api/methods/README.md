# Методы

Base URL:

```
https://proxy.killa.cc/api/v1
```

## Как правильно пользоваться (логика)

Большинство покупок делаются в 2 шага:

1. **Quote** — посчитать итоговую стоимость и квоту _без списания_
2. **Buy / Renew / Prolong** — выполнить операцию _со списанием средств при успехе_

> Quote-эндпоинты нужны, чтобы ты заранее увидел итоговую цену **ровно как в боте** и мог проверить, что баланса хватает.

## Разделы

* **Служебные** → баланс
* **Серверные** → серверные прокси (quote/buy/prolong/protocol)
* **Премиум** → трафик + генерация прокси-листов + whitelist + пароль
* **VPN** → подписка (quote/buy/renew/info)

Переходи к нужному разделу слева в меню.

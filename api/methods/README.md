# Методы

## API разделён на 3 группы

***

### Серверные

Серверные прокси: получение цены, покупка, продление и управление купленными прокси.

***

### Премиум

Премиум-прокси: трафик (цена/покупка/баланс), локации, генерация прокси-листа, whitelist и управление паролем sub-user.

***

### VPN

VPN-подписка: получение цены, покупка, продление и получение информации о подписке.

# Авторизация

API доступен **только по пользовательскому токену**.

## Как передавать токен

Во всех запросах добавляй заголовок:

```http
Authorization: Bearer YOUR_TOKEN
```

## Где взять токен

В Telegram-боте: **Профиль → API** → скопируй токен.

## Сброс токена

В **Профиль → API** нажми **Сбросить токен**.

* Старый токен перестаёт работать сразу.
* Новый токен нужно подставить в интеграцию.

## Безопасность

* Не передавай токен в URL/логах.
* Храни токен в переменных окружения/секрет-хранилище.
* Если токен утёк — сбрось его в боте.

## Пример запроса

```bash
curl -s https://proxy.killa.cc/api/v1/balance \
  -H "Authorization: Bearer YOUR_TOKEN"
```

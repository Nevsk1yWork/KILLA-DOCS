# Введение

**KILLA PROXY API** — единый HTTP API для покупки и управления услугами из бота: **Dedicated** (серверные прокси), **Premium** (трафик/генерация прокси-листов) и **VPN**.

## Для кого и зачем

| Для кого              | Что даёт                                             |
| --------------------- | ---------------------------------------------------- |
| Реселлеры             | Интеграция покупки/продления в свою панель/бота      |
| Команды/автоматизация | Скрипты для quote → buy → управление услугами        |
| Пользователи          | Управление услугами через API вместо ручных действий |

> **Важно:** доступ к API идёт строго в рамках аккаунта владельца токена. Любые покупки/списания выполняются от имени этого пользователя.

## Доступ и ссылки

```
Base URL: https://proxy.killa.cc/api/v1
Документация: https://docs.killa.cc
Авторизация: Authorization: Bearer <token>
Формат: JSON only
```

## Где взять токен

Токен доступен в боте: **Профиль → API**.\
Там же есть кнопка **Сбросить токен** — после сброса старый токен перестаёт работать.

<details>

<summary><strong>Термины (коротко)</strong></summary>

* **Dedicated** — серверные прокси с фиксированными IP (покупка/продление/смена протокола).
* **Premium** — трафик и генерация прокси-листов по параметрам (pool\_type, гео, sticky/rotating и т.д.).
* **VPN** — подписка (покупка/продление) и получение текущей информации.

</details>

## Общий принцип работы

1. Проверяешь баланс.
2. Делаешь **quote** (расчёт стоимости) для нужного сценария.
3. Делаешь **buy/renew/prolong**.
4. Получаешь данные услуги в ответе API (в том виде, как они выдаются в текущей логике сервиса).
